# Zoo Park Navigation

A clean, accessible HTML prototype for a zoo park website navigation and informational pages.

## Overview

This repository contains a lightweight, semantic HTML prototype demonstrating clear navigation and content structure for a zoo or park website. It is intended as a starting point for public-facing informational pages, wayfinding, and simple microsite layouts that prioritize accessibility and maintainability.

## Key Features

- Semantic, standards-compliant HTML5 structure
- Simple, readable markup suitable for learning or rapid prototyping
- Lightweight — no external frameworks required
- Accessible markup and clear navigation patterns
- Easy to host on any static-file server or GitHub Pages

## Demo / Preview

To preview locally, open `index.html` in any modern web browser. For a local development server:

- Python 3: `python -m http.server 8000` then visit `http://localhost:8000`
- Node.js (http-server): `npx http-server . -p 8000`

## Built With

- HTML5

## Getting Started

To get a local copy of the project:

1. Clone the repository:
   git clone https://github.com/seshasai164/zoo-park.git

2. Change into the project directory:
   cd zoo-park

3. Open `index.html` in your browser or run a local server (see Demo section).

## Project Structure

- `index.html` — main entry and navigation
- `*.html` — supporting pages (exhibits, tickets, contact, etc.), if present
- `assets/` — images, stylesheets, and scripts (if added)

Adjust the structure to match your needs as the project grows.

## Usage

This project can be used as:
- A template for zoo or park informational websites
- A prototype for testing navigation, layout, and accessibility
- A learning resource for semantic HTML and simple static site structure
